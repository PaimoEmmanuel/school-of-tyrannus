# School of Tyrannus

## Getting Started

Create a .env.loval file and add this in it:

```bash
NEXT_PUBLIC_API_BASE_URL=https://apps.nlwc.church/tyrannus/api
```

## Note

First, install the dependencies:

```bash
npm i
```

Second, run the development server:

```bash
npm run dev
```

To build the project, run:

```bash
npm run build
```
