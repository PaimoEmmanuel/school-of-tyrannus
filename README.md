# School of Tyrannus

## Getting Started
Create an .env.loval file and add this in it:

```bash
NEXT_PUBLIC_API_BASE_URL=http://67.205.128.129/tyrannus_webapi/api
```
## Note

First, install the dependencies:

```bash
npm i
```

Second, run the development server:

```bash
npm run dev
```

To build the project, run:

```bash
npm run build
```
